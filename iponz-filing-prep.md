# IPONZ Trademark Filing Prep — "Terrified Toys"

**Correction before anything else:** the task brief specified Class 28 and Class 31. Checked the actual Nice Classification (the international system IPONZ uses) directly — **Class 31 doesn't fit this product.** Class 31 covers "agricultural products, live animals, foodstuffs and fodder for animals, bedding and litter for animals" — i.e. pet *food* and animal feed, not toys. **Class 28 ("games, toys and playthings") is the correct and sufficient class for plush dog toys.** Recommend filing Class 28 only, unless there's a concrete near-term plan to sell pet food/treats under the same brand (in which case Class 31 would make sense to add now rather than file twice later) — flagging this as a decision for Daniel rather than assuming.

Re-run the search at [iponz.govt.nz/get-ip/trade-marks/search/](https://www.iponz.govt.nz/get-ip/trade-marks/search/) immediately before filing, since someone else could have filed in the gap.
- [ ] **Decide the mark type:** word mark ("Terrified Toys" as plain text) vs. a combined mark (text + the logo lockup from `brand-strategy.md` Section 4). A word-only mark is cheaper to get right and protects the name regardless of how the logo evolves; a combined mark also protects the specific visual lockup but is narrower. **Recommendation: file the word mark first** (the name is the more durable asset at this stage — the logo/lockup is explicitly described as "interim" in `brand-strategy.md`), and consider a combined-mark filing later once the visual identity is more settled.
- [ ] **Decide on Class 28 only, or 28 + 31** (see correction above).

## 4. Goods & Services Description (Class 28)

Recommended description, using IPONZ's Trade Mark Specification Builder pre-approved terms where possible (this gets the $70/class discount rate instead of $100 — see Costs below):

> Toys for pets; plush toys; soft toys; chew toys for pets; playthings for animals.

This covers the actual product (plush dog chew toys) without over-claiming into unrelated toy categories (e.g. don't claim "video game apparatus" or "sporting goods" just because they're also in Class 28 — narrower, accurate descriptions are both cheaper to defend and less likely to draw an examiner's objection).

**If adding Class 31** (only if pursuing pet food/treats soon): *"Foodstuffs for animals; edible chews for pets"* — again, keep it narrow to what's actually planned, not aspirational.

## 5. Filing Instructions — Step by Step

1. **Log in to IPONZ** at [iponz.govt.nz](https://www.iponz.govt.nz) using RealMe.
2. **Re-run the trade mark search** (Trade Mark Check tool) to confirm "Terrified Toys" is still clear.
3. **Choose the Form Builder** (simpler option for a first-time applicant; the Case Management System has more advanced options but isn't needed for a straightforward single-mark filing).
4. **Enter applicant details** using the template in Section 2 above.
5. **Enter mark details:** select "word mark," enter "Terrified Toys" as plain text.
6. **Build the specification:** use the Trade Mark Specification Builder, select Class 28, and use the pre-approved terms from Section 4 above where they match exactly (using pre-approved terms unlocks the discounted fee).
7. **Review and submit.** Pay the filing fee (see Costs below) — payment is required at submission, you'll receive an application number by email immediately after.
8. **Wait for examination** — IPONZ aims to respond within 15 working days for straightforward applications (longer if there are any Māori-language/cultural elements to assess, which doesn't apply here).
9. **Respond to any examiner queries** if raised (e.g. a request to narrow the specification) — there's a response deadline, don't let this lapse.
10. **Acceptance → 3-month advertisement period** in the IPONZ Journal, during which anyone can oppose the application.
11. **Registration**, typically around 6 months from filing if nothing is contested.

## 7. What to Expect — Timeline

- **Filing → initial examination response:** around 15 working days for a clean application (no Māori-element review needed here)
- **If queries are raised:** add however long it takes to respond + a further examination cycle
- **Acceptance → advertisement period:** 3 months (statutory, can't be shortened)
- **Advertisement → registration** (if unopposed): typically brings total time to **around 6 months from filing**, consistent with IPONZ's own published timeframes
- Opposition (rare for a name like this with no obvious conflicting prior mark) would extend the timeline significantly — not expected here based on the clear search result, but worth knowing it's possible
